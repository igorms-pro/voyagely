This directory contains all design-related documentation for Voyagely.

## Documents

- **[Design System](./design-system.md)** - Complete design system, components, and UI/UX guidelines

## Design Principles

1. **Mobile-First** - Design for mobile constraints first, enhance for larger screens
2. **Clarity & Simplicity** - Clear visual hierarchy, minimal cognitive load
3. **Accessibility First** - WCAG 2.1 AA compliance
4. **Performance** - Fast, optimized user experience
